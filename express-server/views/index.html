Portfolio

Projects

ByteSized Notes

This web application acts as a dashboard for all of your classroom note taking. The information is saved to the user's device, allowing them to open their notes with or without an internet connection. The web application makes use of a service worker to store files to the cache, allowing offline access.

ByteSized Budget

This is a web application for keeping track of your budget. The app will provide stats about your spending back to you.

ByteSized Pantry

This website is a web application in which the user can create an inventory of everything stored in their pantry, add recipes, and create grocery lists. When the user wants to make a meal during the week, the app will gather a list of ingredients they need that they do not already have.

Math Quiz

This website uses vanilla website technologies to allow a player to submit answers to simple addition problems. Each correct answer awards the player a point. The website makes use of a service worker and the Cache API to allow offline availability of the website. This was the first website project to enable this capability.

ByteSized Crew Website and Server

The ByteSized Crew website uses vanilla website technologies and is hosted on a website Server owned and administered by me. All websites created by me are hosted on this website server.
