Notion Experiments

If I didn't mention already, I'm a Notion diehard. Below are some of the experimental uses for Notion I have invented.

Templates

Automated Grocery List

This web application utilizes a database of ingredients (left image) with a stock property. Ingredients listed out-of-stock will appear on the grocery list page (right image) using filters.
Built within Notion. Click either image to open the template right in your web browser.
An independent version of the web app using React JS is in progress.

Fiction Beat Sheet Outliner

This template is designed for outlining a story with accordance to Blake Snyder's Beat Sheet.

If you like these templates, you can support me by using my affiliate link when beginning a Notion account or upgrading to a paid tier.
